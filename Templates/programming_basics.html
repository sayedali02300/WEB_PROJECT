What is Programming?

Programming is the process of writing instructions for a computer to follow. These instructions, known as code, are written in programming languages like JavaScript, Python, C++, and many more.

1. Variables

A variable is a container for storing data values. In most programming languages, you define variables to store things like numbers, text, or objects. 

let age = 25;
const name = "Alice";

2. Data Types

Common data types include:

  • String – Text, e.g., "Hello"
  • Number – e.g., 123, 3.14
  • Boolean – true or false
  • Array – A list of values
  • Object – A collection of key-value pairs

3. Conditional Statements

Conditional statements let your code make decisions. The most common is if. 


if (age > 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

4. Loops

Loops repeat a block of code multiple times. Here's an example of a for loop: 


for (let i = 0; i < 5; i++) {
  console.log("Count: " + i);
}
